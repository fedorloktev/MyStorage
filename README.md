# MyStorage
Program for save some products in some storages
Программный продукт для учета товара и других сущностей на виртуальных складах.

Доброго времени суток Данный файл является инструкцией по использованию программного обеспечения "MyStorage".
Если вы скачали данный архив из репозитория github, то вот краткое описание иерархии скачанных вами файлов:

1) foldReadyJARfiles - папка с готовыми файлами программы в формате .JAR для запуска
2) out - папка содержит скомпилированные файлы классов
3) src - папка содержит исходные данные классов в формате .java

Техническая информация:
Для развертывания данного ПО на вашем компьютере не понадобится наличие сервером Баз Данных и какие-либо установки. 
Программа не использует допольнительного программного обеспечения что облегчает ее использование.
Хранение входных данный происходит с помощью файлов JSON формата.
Выходные данные реализуются в HTML формате.

Инструкция по запуску:
Для того, чтобы начать использовать программное обеспечение, необходимо из корневого каталога зайти в папку 
"foldReadyJARfiles" и двойным щелчком мыши запустить файл "startApp.jar".

Инструкция по использованию программного обеспечения:
1) Окно авторизации. В данном окне вам необходимо ввести логин и пароль для входа в программу. Для ознакомительных
целей вы можете использовать логин: admin и пароль также admin после чего вам необходимо нажать кнопку SignIn для входа.
Если логин и пароль были введены корректно - вы попадете в следующее окно, если нет - вам сообщит об этом система
выведя текстовое сообщение (красным цветом под полями ввода).
2) Окно просмотра складов. Оно же основное меню программы. В данном окне вы можете выбрать один из 4-х существующих
складов (название складов задано в ознакомительных целях) для просмотра содержащихся на складе товаров. Также вы можете
по нажатию кнопки "Добавить" осуществить добавление помого продукта на один из существующих складов (см. "Окно добавления").
Для непосредственного учета товарооборота в правой стороне окна существуют кнопки, которые в формате HTML-файлов
покажут необходимые движения товаров на складах.
3) Окно добавления. С помощью данного окна вы можете добавить новый продукт на ваш склад. Для этого необходимо выбрать
один из существующих складов с помощью выпадающего меню, после этого указать название товара без использования кавычек "
если таковые будут использованы - вас оповестит об этом система и попросит заменить данный символ. После этого необходимо
ввести артикул товара, который может состоять из букв и цифр, а так же цену товара, которая должна быть указана в 
числовом значении в виде целого или дробного числа. При использовании букв и других символов вам будет показано сообщение.
После того как все поля заполнены, вы можете нажать кнопку добавления товара и при успешном добавлении система вас об этом
обовестит.
4) Окно просмотра. Для того, чтобы просмотреть содержимое на складе, вам нужно выбрать и щелкнуть на один из 4-х складов.
Откроется окно просмотра в котором с левой стороны будет список из имеющихся на складе товаров. Если их нет, система выдаст
сообщение еще до перехода на страницу склада. На данной странице вы можете просто посмотреть наличие товаров, либо
совершить с ними какие то действия. Для этого необходимо выбрать нужный товар и воспользоваться одной из кнопок:
4.1) Кнопка просмотра. При нажатии на данную кнопку вам будет продемонстрирована детальная информация по товару.
4.2) Кнопка продажи. При нажатии на кнопку вы сможете "продать" выбранный товар. Для этого необходимо указать 
стоимость и количество продаваемых экземпляров. В ознакомительном варианте отсутствует фактическое количество товара, 
поэтому можно вводить любую сумму без ограничений. После чего нужно нажать кнопку продажи, при успешной продаже система
оповестит вас об этом.
4.3) Кнопка смены склада. Если вы ошиблись при добавлении товара и направили его не на тот склад, вы можете изменить это
нажав на данную кнопку. При нажатии откроется окно, в котором необходимо задать склад, на который вы хотите переправить
товар. После чего нажать кнопку подтверждения, система оповестит вас о добавлении товара на склад. После данной операции
товар окажется на другом складе, удалившись из списка этого склада.
5) Группа окон отчетности. При выборе одной из кнопкок отчетности вам покажется информация о выбранных операциях в виде
HTML файлов, открытых в стандартном для вашей системе обозревателей файлов данного формата.
